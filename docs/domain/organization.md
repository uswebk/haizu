# 組織（会社・事業所）

## 概要
haizu に登録する会社・事業所の単位。全データの最上位エンティティ。

## 用語
- 組織
    - サインアップ時に会社名を入力することで作成される
    - 1メールアドレスにつき1組織
- 事業所名
    - 組織の名称。設定画面から変更可能
- 組織管理者
    - サインアップしたユーザーが最初の「管理者」として自動設定される

## ルール
- 1組織に拠点を最大10個まで登録可能
- 1組織に複数のメンバー（ログインユーザー）を招待可能
- 組織の削除は行わない（非アクティブ化のみ）
- 事業所名・メールアドレスは「事業所情報」設定画面から変更可能

## 関連ドメイン
- 組織 1 : N 拠点
- 組織 1 : N メンバー（member_permission.md 参照）
