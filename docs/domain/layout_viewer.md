# 配置Viewer
## 用語
- 自動更新(Phase2対応)
    - 指定した時間ごとに画面をリフレッシュする
- 自動切り替え(Phase2対応)
    - 複数画面をまとめて切り替えて表示する
- サイズ調整
    - ディスプレイのサイズに合うように調整可能(拡大・縮小)
    - ブラウザ全画面表示することも可能
## ルール
- 配置ビュアーは閲覧用画面であり、操作はできない
- 自動更新設定により、自動的に最新の配置を取得するようにできる
- 自動切り替え設定により複数のエリアを切り替えながら表示することができる
    - モニターが1つしかないケースを想定
## その他
- ビュアーに表示する内容をコントロールできる
    - シフトに応じて流動的に変える
    - 強制表示(作業が伸びた場合など)
    - ビュアー設定
        - エリアを選択
            - シフト時間に合わせて表示
            - シフトのx分前(後)に表示
            - 強制表示
                - 日付+シフトを選択
        - アナウンス設定
            - 指定時間内でアナウスを文字で表示する
                - 内容
                - 開始日時~終了日時
