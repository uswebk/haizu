# 従業員
## 用語
- 名前
    - 姓・名を登録する
- 従業員コード
    - 従業員識別のための管理用のコード
    - 重複不可
- アバター
    - 従業員ごとにカラーコードを設定する
- 管理タグ
    - 従業員のフィルター用のタグ
    - 拠点ごとにタグを作成し、従業員に紐付けることができる
        - 「役職」「作業内容」などを紐づけて識別に使用する
    - タグは拠点単位で管理する（他拠点とは共有しない）
    - 1従業員に複数タグを付与可能
    - 配置決め画面でのフィルタリングに使用する（未配置プールの絞り込み）
- 入社日
    - yyyy-mm-dd で任意で登録する
    - 経験年数として使用する
- 退職日
    - yyyy-mm-dd で任意で登録する(センシティブ情報のため拠点管理者のみ確認できる状態)
## ルール
- 拠点間で共通の従業員は作成できない
    - 同従業員の場合でも拠点ごとに従業員登録が必要
    - 拠点へのコピー機能を作成する
- 拠点内における従業員コードは重複不可。拠点をまたがるのはOK
- 従業員削除
    - どの配置にも紐づかない従業員は削除可能。
    - 紐づいている場合は、非表示設定となる
