# 拠点
## 用語
- 拠点
    - 「A食品工場」「B出荷倉庫」など
- 働き方
    - 拠点ごとに働き方を登録する
    - 日勤/遅番/夜勤 など
    - この設定により、配置決めに使用するシフトが決定される
- 拠点選択
    - メンバーがどの拠点の操作を行うか選択する
    - 拠点に招待されているメンバーのみが閲覧可能
## ルール
- 会社ごとに10個まで拠点登録可能
    - 複数拠点を持つ会社に対応
- 1会社で1つ以上の拠点を持つ
    - 拠点がないとサービス利用ができない
- 働き方の設定変更
    - 途中で変更可能だが、過去のデータは過去のシフトが紐づく
- 拠点は削除できない、非アクティブ設定は可能
- 拠点の作成・編集ができるのは「管理者」のみ
    - 「拠点管理者」以下の権限では拠点の新規作成・編集は不可
- 拠点に紐づくデータ（配置エリア、従業員、配置決め、勤務体制、タグ、Viewer設定など）へのアクセスも、拠点選択と同様「管理者」または当該拠点に招待されているメンバーのみ可能
