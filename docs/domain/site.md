# 拠点
## 用語
- 拠点
    - 「A食品工場」「B出荷倉庫」など
- 働き方
    - 拠点ごとに働き方を登録する
    - 日勤/遅番/夜勤 など
    - この設定により、配置決めに使用するシフトが決定される
- 拠点選択
    - メンバーがどの拠点の操作を行うか選択する
    - 拠点に招待されているメンバーのみが閲覧可能
## ルール
- 会社ごとに10個まで拠点登録可能
    - 複数拠点を持つ会社に対応
- 1会社で1つ以上の拠点を持つ
    - 拠点がないとサービス利用ができない
- 働き方の設定変更
    - 途中で変更可能だが、過去のデータは過去のシフトが紐づく
- 拠点は削除できない、非アクティブ設定は可能
