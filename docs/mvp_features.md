- 認証認可
  - ユーザーはemail, password を使用して当サービスに登録・ログインする
  - OTPをemailに送信し本人確認を行う
  - 登録時に使用する項目: email, password, 会社名
- 拠点登録
    - 複数拠点ある場合も1事業所で対応できるようにする
        - A工場, B倉庫, Cマンションなど
- 従業員登録
  - 該当拠点で働く従業員を登録する
      - 手動での登録、CSVを用いた一括登録
      - オプショナルでそのた情報も設定もできるように(年齢、入社日、退職、役職, アバターカラー)
- 配置規格登録
    - フロアマップ画像アップロードの行う(任意)
    - 配置登録(どこに人員を設置するか)
        - ドラッグ&ドロップで手軽に移動
    - 規格のバージョン管理
- 配置割り当て
    - 日付を選択(日夜勤設定も、3交代制まで -> これは設定で変えれるように)
    - 配置規格に対して従業員をアサイン
        - ドラッグ&ドロップで手軽に移動
        - 配置を押して従業員を設定
- 配置履歴
    - 日付を指定して過去の配置実績を確認
    - メモを残せるように、CSVエクスポートできるように
- 配置ビュアー
    - モニターなどに表示する用
    - 操作はできない
    - 管理者からのアナウンスなどもここに表示する
- 管理者用設定画面
    - 配置を選んでアナウンスなどを入力する(時間指定可能)
- 設定
    -  アカウント設定
        - 事業所名
        - メールアドレス
    - 権限設定
        - 管理者: 拠点を跨いでなんでもできる
        - 拠点管理者: 拠点内でなんでもできる
        - 一般: 配置ビュアーの確認、配置履歴の確認ができる
        - その他: 配置ビュアーの確認だけができる
    - 働き方
        - 1交代
            - 何時~何時
        - 2交代
            - 何時~何時
        - 3交代
            - 何時~何時
